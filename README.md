# exercise-eda-r
Exploratory Data Analysis of 973-observation exercise dataset using R (RStudio)

**Overview:**
- Visualised distributions and relationships using histograms, boxplots, scatterplots, bar charts, and heatmaps
- Cleaned implausible values and examined missingness
- Analyzed key relationships, including workout duration vs. calories burned
- Delivered results in a structured statistical report

**Tools:**
- RStudio, ggplot2, dplyr

**Files:**

src/ — R scripts

report/ — PDF report
